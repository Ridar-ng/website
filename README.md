# Stika Landing Page

This is the landing page for Stika, a Nigerian mobile advertising platform that connects keke (tricycle) riders with advertisers, allowing businesses to reach millions on Nigerian streets.

## Features

- Responsive design optimized for all devices
- Interactive elements including:
  - Tabbed content sections
  - Testimonial slider with navigation
  - Interactive timeline for the user journey
- Clear sections for advertisers, agents, and keke riders

## Technologies Used

- HTML5
- CSS3
- JavaScript (vanilla)
- Tailwind CSS for styling

## Pages

- Home page with sections for:
  - Hero section
  - Statistics
  - Join Network options
  - How it Works timeline
  - Testimonials
  - Disclaimers
  - Footer with app download links

## Setup

Just clone the repository and open index.html in your browser to view the website.
